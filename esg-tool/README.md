# 「苏ESG」苏州本土企业ESG量化分析与智能投研平台
## 项目概述
本项目是西交利物浦大学金融数学专业的学术研究与创业实践项目，聚焦苏州本土上市公司与科创企业，融合金融数学核心模型（马科维茨均值-方差模型、相关性分析、夏普比率计算）与ESG投资理念，开发了一套完整的ESG量化分析工具，为苏州企业、投资机构提供专业的ESG评分、风险分析、投资组合优化服务，助力苏州ESG产业创新发展。

## 项目作者
西交利物浦大学 金融数学专业 大一学生

## 技术栈
- **前端框架**：Streamlit 1.32.0
- **数据处理**：Pandas 2.2.1, NumPy 1.26.4
- **可视化**：Plotly 5.19.0
- **量化模型**：马科维茨均值-方差模型、相关性分析、夏普比率计算
- **优化算法**：Sequential Least Squares Programming (SLSQP)

## 核心功能
### 1. ESG综合评分计算器
- 支持自定义E/S/G三个维度的权重，自动计算综合ESG评分与评级
- 生成交互式雷达图，对比公司与苏州本土行业平均表现
- 适配苏州六大支柱产业（电子信息、装备制造、生物医药、新能源、新材料、金融服务）

### 2. ESG与财务表现相关性分析
- 基于苏州本土上市公司真实数据，计算ESG得分与收益率、波动率的相关系数
- 生成散点图直观展示ESG与财务表现的量化关系
- 按行业分组分析苏州本土企业ESG表现差异

### 3. 基于ESG约束的投资组合优化
- 实现经典马科维茨均值-方差模型，支持设置投资组合最低ESG约束
- 计算最优资产权重分配，生成有效前沿曲线
- 计算夏普比率等风险调整收益指标，为ESG投资决策提供量化支持

## 数据来源
- 苏州A股上市公司2024年ESG报告（万得Wind金融终端）
- 苏州统计局公开行业数据
- 苏州ESG产业创新中心公开研究报告
- 西浦国际商学院ESG研究数据库
